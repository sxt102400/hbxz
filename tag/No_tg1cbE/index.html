JDK

JDK各个版本的新特性

** 1. JDK1.5新特性**

  • 枚举
  • 泛型
  • 自动装箱与拆箱
  • 静态导入
  • 可变参数
  • 内省
  • For-Each循环

2. JDK1.6新特性

  • JAXB2实现xml和对象映射
  • 脚本语言的支持
  • 插入式注解
  • Compiler API

** JDK1.7有什么新特性**

  • switch语句支持字符串变量
  • catch代码块捕获多异常
  • try-with-resources自动资源关闭
  • 支持二进制字面量，"0b"前缀，
  • 数字的"_"连数符，支持数字字面量可读性
  • NIO2.0的Path

4.JDK8有什么新特性

  • 函数式接口
  • Lamb表达式
  • Stream流
  • 新的Date,Time API
  • Nashorn javascript脚本引擎
  • 重复注解，现在你可以将相同的注解在同一类型上使用多次。

JDK 中常用的包有哪些？

答：lang，util，io，net，sql。
